# Screenshots

This folder contains screenshots of the ForceUI application for demo purposes.

## How to Capture Screenshots

1. Start the development server: `cd forceui && npm run dev`
2. Open http://localhost:3000/forceui in your browser
3. Use your browser's screenshot tool or system screenshot utility
4. Save screenshots here with descriptive names:

### Suggested Screenshots:
- `01_welcome_screen.png` - Initial welcome state
- `02_mcp_integrations.png` - MCP Status Panel showing integrations
- `03_workflow_recorder.png` - Workflow Recorder component
- `04_product_planning.png` - Timeline + KanbanBoard for planning
- `05_ai_reasoning.png` - Explainability Panel showing AI decisions
- `06_persona_switch.png` - Persona switcher in action

## Tips
- Use browser DevTools (F12) to simulate different screen sizes
- Capture at 1920x1080 for best presentation quality
